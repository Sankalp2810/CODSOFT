# Titanic Survival Prediction

## Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning. A Logistic Regression model is trained on the Titanic dataset after performing data preprocessing and exploratory data analysis.

## Dataset
- Dataset: Titanic Dataset
- Target Variable: Survived
- Features include:
  - Pclass
  - Sex
  - Age
  - SibSp
  - Parch
  - Fare
  - Embarked

## Project Workflow
1. Import libraries
2. Load dataset
3. Data preprocessing
4. Handle missing values
5. Exploratory Data Analysis (EDA)
6. Encode categorical variables
7. Split dataset into training and testing sets
8. Train Logistic Regression model
9. Evaluate model accuracy

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Training Accuracy: 80.76%
- Testing Accuracy: 78.21%
